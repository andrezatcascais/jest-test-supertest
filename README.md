## How to run

###### *1 - Install dependencies*

```bash
npm install
```

###### *2 - Build the development server*

```bash
npm run build
```

###### *3 - Start the development server*

```bash
npm run dev
```

###### *4 - Start the unit test*

```bash
npm run unit-test
```

###### *5 - Start the integration test*

```bash
npm run integration-test
```

###### *6 - Start the all test*

```bash
npm run test
```
